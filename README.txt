Курсовой проект. 
Тема: веб-приложение, моделирующие работу с системой умного дома
Технологии: Express.js, mongodb, ajax, ejs

- регистрация/авторизция
- профиль польщователя + профиль дома
- выбор планировки дома
- добавлнеи, изменение, удаление устройств
- "включение/выключение" устройств
- фильтрация устройств по комнатам и по типу (функционал)
- динамическое создание персональных страниц устройств
- комментирование

Для запуска данного поректа необходимо наличие node.js server и mongodb.

Первое что нужно сделать - это запустить базу данный. Для этого необходимо запустить консоль
(например, сочетанием клавиш Win+R и ввести в поисковую строку cmd). Затем через консоль кудно перейти
в папку базы данных (cd C://mongodb/bin) и, наконец, запустить базу данных командой mongod.

При этом база данных добжна назваться SmartHome и иметь коллекции comments, devices, users.
В коллекции users должен лежать хотя бы один документ, хранящий login И password пользователя.

Структура документов коллекций
users: login, password, phone, email, homeLogin, template
devices: login, userLogin, room, type, condition
comments: login, text
//более подробную информацию о структуре базы данных можно посмотреть в папке models проекта

Далее, необходимо в командной строке (терминале) ввести команду npm run dev. Данная команда, 
непосредственно, запустит само приложение. 

В браузере его можно открыть перейдя по ссылке http://localhost:3000/
